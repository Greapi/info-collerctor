🏭 美团开源万亿参数大模型 LongCat-2.0，学术成果获 ACL 杰出论文

  • Linux 内核“GhostLock”漏洞：一个栈上释放后使用漏洞在全部 Linux 发行版中存在长达 15 年，现被曝光。来源

趋势与判断

  • 大模型能力评测窗口期缩短，成本与效率成新战场：Grok 4.5 在特定任务上达到头部水平的同时，强调成本效率；GPT-5.6 迁移案例显示降本 27% 与提速 2.2 倍；医疗领域直接对比更突显实用价值。模型间的竞争正从单纯跑分转向生产场景的费用-效果比。参见 Grok 4.5 评测 与 迁移案例
  • AI Agent 生态基建快速成型：从美团 LongCat-2.0 专攻 Agentic Coding，到 GitHub 上 11.9 万星的 AI Agent/RAG 应用集 awesome-llm-apps，再到 Clawk 的安全运行沙箱，以及编程 Agent 危险命令拦截工具 destructive_command_guard，围绕 AI Agent 的开发、部署与安全的全链路工具正在密集涌现。
  • 开源与国产算力协同的新范式：LongCat-2.0 首次在五万卡国产集群上完成万亿参数推理，并提供多精度版本及优化推理代码，显示开源大模型正在与国产算力形成“模型-硬件”闭环，有助于盘活存量算力资源。来源
  • LLM 跳跃式创新仍存想象空间：针对“LLM 无法飞跃”的观点，专业人士以 AlphaGo 为例进行反驳，并指出当前未让 AI 代理进行长期不间断计算，且会话记忆碎片化是瓶颈，暗示未来通过架构调整可能释放新的创新能力。来源：RT Yun-Ta Tsai...
  • AI 安全与隐私冲突升级：Grok CLI 上传整个主目录的事件，将工具便利性与用户隐私的平衡问题推到风口浪尖；同时 Linux 15 年漏洞 GhostLock 被曝，也提醒基础设施安全性需要持续审视。来源 及 GhostLock
