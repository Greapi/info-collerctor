今日概览

今日科技圈焦点密集：SpaceX 完成年度第 17 次拼车发射，并向 FCC 申请运营由 10 万颗卫星组成的第三代星链星座，马斯克称需要更强大的星舰；社交平台 X 以 43.99 亿月访问量位居全球第六，环比增速居前十之首；AI 领域，Grok 视频与语音全面升级，而美国公司加速采购中国模型的同时，中国拟对前沿模型分级出海，地缘技术博弈升温。此外，欧盟“聊天控制”法案首轮通过，微软裁撤 id Software 的 idTech 团队，DeepSeek 宣布将实施峰谷定价，开发者社区围绕 Claude Code、Coding Plan 与 API 安全的讨论持续高涨。

重点事件

  • SpaceX“运输者-17”发射成功，同步推进 10 万卫星星座计划

    • 猎鹰 9 号火箭第 17 次“运输者”拼车任务将 81 个有效载荷送入轨道，火箭一级成功回收着陆。同日，SpaceX 正式向 FCC 申请发射并运营 10 万颗卫星的第三代星座，马斯克回应称“我们需要更大的火箭！（星舰）”。 Falcon delivers 81 satellites to orbit | We’re gonna need a bigger rocket! | RT SpaceX: Falcon 9 landing confirmed

  • X 平台 6 月访问量全球第六，环比增长领先

    • 据 Similarweb 数据，X（原 Twitter）2026 年 6 月全球访问量 43.99 亿次，环比增长 4.66%，为全球前十大网站中增速最高，已超过 Reddit 和 TikTok。 𝕏 had the highest monthly growth of any top 10 site in June

  • Grok Imagine 支持 15 秒视频生成，Grok 语音新增 21 种旗舰音色

    • Grok Imagine 更新后可生成极高质量的 15 秒视频。同时 SpaceXAI 发布 21 种全新多语言旗舰语音，覆盖超 25 种语言，原有 5 种语音的节奏与自然度也获提升，开发者可通过 API 构建自定义语音代理。 Grok Imagine update | Grok Voice upgrade

  • 欧盟议会首轮通过“聊天控制”法案

    • 备受争议的 Chat Control 法案在欧盟议会首轮投票中过关，可能对加密通信与隐私保护产生深远影响。 Chat Control passed first round in EU Parliament | Chat Control 1.0 and 2.0 Explained

  • 美国公司加速采用中国 AI 模型，中国拟对前沿模型分级出海

    • CNBC 报道，因 OpenAI、Anthropic 成本上升，DeepSeek、GLM 5.2 等中国开源模型性能仅落后约 6-9 个月，但成本低 60%-90%；通过 OpenRouter 采购中国模型的美国公司占比已从 11% 升至 30%-46%。与此同时，路透社称中国商务部约谈阿里、字节、智谱等，计划限制最先进模型（含开源权重）的海外访问，实行分级管理。 美国公司开始更多采用中国 AI 模型 | 中国可能要开始给前沿 AI 模型分级出海

  • 微软裁撤 id Software 的 idTech 引擎开发团队

    • 据报道，微软已解雇 id Software 旗下 idTech 引擎团队，具体影响尚未明朗。 Microsoft fire idTech team at Id software

  • DeepSeek API 将启动峰谷定价

    • DeepSeek 预计 7 月中旬实施分时价格，高峰时段（北京时间 9:00-12:00、14:00-18:00）收费为平日的 2 倍。 deepseek 居然出了峰谷价格

  • 苹果 iPhone Ultra 折叠屏前瞻：4:3 比例，展开即 iPad 体验

    • 分析指出 iPhone Ultra 采用 4:3 比例，灵感或与 WWDC26 相关；通过 Xcode 27 可查其逻辑分辨率，体验被描述为“合上是 iPhone，展开是 iPad”。 iPhone Ultra 软硬件超前瞻

趋势与判断

  • AI 模型流动双向加速，地缘技术监管成形

    • 中国企业模型因极致性价比吸引海外客户，但国内监管部门已着手构建“基础开源备案—高级安全审查—最前沿仅限国内”的分级出海框架。技术开源与地缘管控之间的张力将成为下一阶段 AI 产业的关键变量。

  • AI 开发工具生态高度内卷，开发者面临范式重构

    • Claude Fable5 即将下线、Cursor 用户转投 Claude Code、国内 Coding Plan 与 API 中转站竞相低倍率推广等现象，折射出模型能力跃升下开发工作流的剧烈变动。同时，API Key 盗刷、模型消耗异常等安全问题频发，进一步加剧开发者对新工具“又爱又恨”的复杂心态。

  • 欧盟监管加压，平台与隐私博弈升级

    • Chat Control 法案首轮过关，若最终落地，将根本性改变欧洲甚至全球的即时通讯加密实践。结合此前科技巨头突然对“AI 消灭工作”改口，技术与制度的摩擦面正在扩大。

  • 太空基础设施建设提速

    • 从 SpaceX 高频拼车发射到申请十万卫星星座，再到明确需要星舰提供运力支撑，低轨空间资源竞争正进入新阶段，也为全球连接和 AI 分布式算力铺设了想象空间。
