# 2026-07-16 RSS 日报

## 今日概览
今日技术社区聚焦于 Kimi 旗舰模型 K3 正式上线及会员权益争议；马斯克旗下 SpaceX 星舰 Flight 13 已堆叠准备飞行、Grok Build 开源；OpenAI 回应 GPT-5.6 文件删除问题。开源及产品领域：微软 Comic Chat 开源、OnePlus 暂停欧美业务、Google NotebookLM 更名 Gemini Notebook。

## 重点事件
- **Starship Flight 13 堆叠完毕**：Starship Flight 13 已于今日完成堆叠并准备飞行。[来源](https://x.com/elonmusk/status/2077801061343981703) [来源](https://x.com/elonmusk/status/2077748491045024212)
- **xAI Grok Build 正式开源**：包含 CLI 工具及基于 Unicode 的 Mermaid 图表终端渲染器，代码总量 84.4 万行 Rust，本地优先运行，默认关闭数据保留。[来源](https://x.com/elonmusk/status/2077748220931854467) [来源](https://x.com/elonmusk/status/2077558358429569135)
- **Kimi K3 正式上线**：旗舰模型 K3 拥有 1M 上下文窗口，但在 Kimi Code 中仅 199 元/月以上档位可使用 1M 上下文（99 元/月限制 256k），客户端中仅 699 元/月最高档可使用完整 1M 上下文，引发社区对会员权益分级的热议。[来源](https://www.kimi.com/en) [来源](https://www.v2ex.com/t/1227856#reply14)
- **OpenAI 回应 GPT-5.6 意外删除文件**：问题主要发生在启用“完全访问模式”且无沙箱保护下运行 Codex 时，模型可能错误覆盖 `$HOME` 环境变量导致误删。OpenAI 表示将通过更新开发者消息、引导安全权限模式及增加附加保护措施降低风险，即将发布详细事后分析。[来源](https://x.com/sama/status/2077810840309510550)
- **OpenAI 强调青少年安全使用 AI**：公司介绍了适龄保护、学习工具、家长控制和专家合作等措施，让 ChatGPT 对青少年更安全。[来源](https://openai.com/index/why-teens-deserve-access-safe-ai)

## 趋势与判断
- **AI 模型订阅分层加剧**：Kimi K3 的会员权益与模型能力深度绑定，高端功能仅最高档可用，反映出 AI 厂商正加速通过订阅分层实现商业变现。Kimi 并非个例，Codex 用户也在讨论拼车分摊订阅成本。[来源](https://www.v2ex.com/t/1227856#reply14) [来源](https://www.v2ex.com/t/1227861#reply3)
- **AI 安全与失控风险并存**：GPT-5.6 文件删除事件暴露了 AI 在无沙箱环境下的潜在破坏性；与此同时，SpaceXAI 起诉利用 Grok 制作非法 Deepfakes 的用户，2026 年已提交 73,604 份报告协助至少 244 次逮捕。安全治理与风险管控成为行业焦点。[来源](https://x.com/sama/status/2077810840309510550) [来源](https://x.com/elonmusk/status/2077603867613655101)
- **轻量化开源 AI 工具崛起**：Grok Build 开源、Grok 4.5 在 FrontierSWE 基准中排名第二，以及社区涌现大量开源项目（如 LeAgent 全栈 Agent 平台、CodeDrobe Theme 换肤工具、mdsweep 清理工具等），显示开发者生态正向开放、可定制方向快速演进。[来源](https://x.com/elonmusk/status/2077558358429569135) [来源](https://x.com/elonmusk/status/2077392279552012346) [来源](https://www.v2ex.com/t/1227871#reply0)
- **硬件与基础设施变革信号**：ASML 2027–2028 年产能扩张已考虑到马斯克德州“Terafab”芯片设施需求，EUV 产能接近订满；FreeBSD 16 移除最后一段 GPL 代码实现完全无 GPL，基础软件生态持续演进。[来源](https://x.com/elonmusk/status/2077410816052154428) [来源](https://www.phoronix.com/news/FreeBSD-16-Goes-GPL-Free)
