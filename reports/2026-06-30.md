### Claude Code 被曝嵌入“间谍软件”级环境指纹
社区爆料称 Claude Code 自 `@anthropic-ai/claude-code@2.1.91` 起，会根据自定义 `ANTHROPIC_BASE_URL` 和本机时区改写 system prompt 中的 `currentDate`，被指专门检测中国用户和 API 网关。[来源](https://www.v2ex.com/t/1224016#reply42) | [Reddit 讨论](https://www.reddit.com/r/ClaudeAI/comments/1ujila1/anthropic_embedded_spyware_in_claude_code_and/) | [技术分析](https://random-walk.atoma.one/claude-code-prompt-marker/) | [中文分析](https://dev.com.cn/posts/claude-code-5chk)
- 社区分析认为 Anthropic 大规模封号可能并非只因 IP 问题，Claude Code 客户端存在用户环境指纹泄露风险。[来源](https://www.v2ex.com/t/1224025#reply14)
- 已有用户分享通过 Docker Sandbox 运行 Claude Code 并指定时区的规避方法。[来源](https://www.v2ex.com/t/1224056#reply1)
- 另有用户反映 Claude Code 在 opus 4.8 模型、思考级别 max 时出现严重幻觉，虚拟提交、推送和创建 PR 均为凭空编造。[来源](https://www.v2ex.com/t/1223775#reply3)

## 趋势与判断

### Claude Code 信任危机可能加速开发者迁移
Claude Code 被曝通过 steganographic 技术标记请求、检测用户时区和代理 URL 的行为，[技术分析](https://thereallo.dev/blog/claude-code-prompt-steganography) 已在开发者社区引发强烈反应。Anthropic CEO Dario Amodei 关于“AI 开源是伪命题”的言论更激化了矛盾，博主阮一峰透露其 Anthropic 账户因“违反政策”被暂停，认为存在歧视性政策。[阮一峰博客](http://www.ruanyifeng.com/blog/2026/06/anthropic.html) 结合社区涌现的规避方案（Docker Sandbox）、防御性配置体系（cc-use-exp）和 API 中转方案，用户信任受损可能加速部分开发者向其他平台迁移。

### 美国政治经济结构性调整信号密集
最高法院削弱联邦监管结构、USAID 资金争议（削减后非洲死亡率未增反降）、2026 大选加密行业投入 1.89 亿美元 [Reuters](https://www.reuters.com/world/crypto-firms-have-spent-189-million-so-far-2026-us-election-report-says-2026-06-30/)、劳动力收入占比处于战后最低水平 [纽约联储](https://libertystreeteconomics.newyorkfed.org/2026/06/the-post-covid-decline-in-the-labor-share/) 等信号密集出现，显示美国制度层面正经历深刻调整。选举诚信争议（最高法院裁决被批为选举欺诈开绿灯）进一步加剧政治对立。

### AI 编码工具生态碎片化加剧
Cursor 发布 iOS 版本支持云代理构建 [X](https://x.com/elonmusk/status/2071650759330906253)，Google 发布 agents-cli 工具将编码助手转变为 Google Cloud AI 智能体专家 [GitHub](https://github.com/google/agents-cli)，外加 Claude Code、Codex 等工具，开发者面临工具选择碎片化。火山方舟 Coding Plan 用户反馈显示模型“降智”可能受上下文膨胀、缓存等多种因素影响。[来源](https://www.v2ex.com/t/1223789#reply10)
