# 2026-05-29 RSS 日报

## 今日概览
AI 模型密集发布、AI 编程成本争议与行业乱象成为今日焦点。xAI、Anthropic、Liquid AI 均有新模型亮相，而 OpenAI 员工披露的旗舰模型月消耗可达 130 万美元，引发对 AI 编程可持续性的广泛讨论。开源工具生态活跃，多款 AI 记忆、可视化及开发辅助项目涌现；同时，企业强推 AI 引发的职场形式主义、宽带限速、VC 下场自研等话题，折射出 AI 热潮下的多重张力。

## 重点事件

- **xAI 发布 grok-build-0.1 模型，专注代理编码**：该模型已通过 xAI API 进入公开测试阶段，专注 agentic coding。API定价为每百万输入Tokens 1美元，输出Tokens 2美元。 [来源](https://x.com/elonmusk/status/2060451655301132662)

- **Anthropic 推出 Claude Opus 4.8**：新版本强调更强判断力与长时间独立工作能力，在代理编码、终端操作等方面提升显著，价格保持不变。但有用户反馈 Token 消耗极快，甚至出现模型自称 Qwen 的异常。 [来源 Claude Opus 4.8 发布](https://www.v2ex.com/t/1216353#reply0) | [Token 计算问题](https://www.v2ex.com/t/1216351#reply0) | [自称 Qwen](https://www.v2ex.com/t/1216588#reply3)

- **Liquid AI 发布 8B-A1B MoE 模型**：该模型在 38T 数据上训练，采用混合专家架构。 [来源](https://www.liquid.ai/blog/lfm2-5-8b-a1b)

- **OpenAI 启动 Rosalind Biodefense 项目**：扩展 GPT-Rosalind 访问权限，面向受信任的开发者与美国政府合作伙伴，推进生物防御与流行病防范工作。同时，波士顿儿童医院利用 OpenAI 技术已帮助诊断超 40 例罕见病。 [来源](https://openai.com/index/strengthening-societal-resilience-with-rosalind-biodefense)

- **特斯拉 FSD 在欧洲再下一城**：继荷兰和立陶宛之后，FSD 在爱沙尼亚获批，成为欧洲第三个批准国。 [来源](https://x.com/elonmusk/status/2060349039099469896)

- **加州州议会通过《保护我们的游戏法案》**：该法案旨在保护消费者对已购数字游戏的持续访问权，防止厂商远程关闭游戏。 [来源](https://www.invenglobal.com/articles/22330/stop-killing-games-movement-gains-momentum-california-assembly-passes-game-protection-bill)

- **白宫提议政治任命官员拥有科研拨款最终审批权**：新规将赋予政治任命官员对研究拨款的最终决定权，引发学界担忧。 [来源](https://www.scientificamerican.com/article/white-house-proposes-new-rules-giving-political-appointees-final-say-on-research-grants/)

## 趋势与判断

- **AI 编程成本引发商业模式拷问**：据 OpenAI 员工案例，程序员不限量使用旗舰模型月消耗 Token 价值可达 130 万美元；即便改用国产开源模型，每人每年也至少花费 200 万至 300 万元。Uber 今年头四个月已花完全年 34 亿美元 AI 预算，微软也因费用超标放弃 Claude Code。AI 编程的高昂成本可能迫使企业限制员工不限量使用外部模型。 [来源](http://www.ruanyifeng.com/blog/2026/05/weekly-issue-398.html)

- **AI 落地出现“形式主义”与信任危机**：一方面，有企业 CTO 强推 AI 要求研发提效 100%，但因生成内容质量低，员工被迫通宵重做后汇报为“AI 提效”；另一方面，Turso 公司因难以应对 AI 生成的虚假漏洞报告和引发的无尽争论，被迫终止漏洞赏金计划。 [企业 AI 乱象](https://www.v2ex.com/t/1216367#reply0) | [赏金计划终止](http://www.ruanyifeng.com/blog/2026/05/weekly-issue-398.html)

- **“AI 记忆”与个人上下文工具成新方向**：Hermes Memory Sidecar v3.0、PAM-OS、OpenContext 等多个项目涌现，旨在为 AI Agent 提供外挂长期记忆或统一的个人行为上下文，反映出多 Agent 协作场景下对状态持久化与个性化记忆的强烈需求。

- **基础设施限速问题凸显**：上海电信 2000M/300M 精品网宽带存在达量限速问题，上行约 1TB 即触发，单 IPv4 限速约 1.1MB/s。Cloudflare 则分享了大规模编排 AI 代码审查的经验，显示基础设施在 AI 时代面临双向挑战。 [上海电信限速](https://www.v2ex.com/t/1216332#reply6) | [Cloudflare AI 代码审查](https://blog.cloudflare.com/ai-code-review/)

- **开源项目生态迁移苗头初现**：CodeMirror 作者将所有仓库在 GitHub 上存档并迁移至自建服务，原因是对 GitHub 长期不满；有开源项目被用于钓鱼攻击，引发对开源滥用风险的新讨论。 [CodeMirror 迁移](https://www.v2ex.com/t/1216539#reply3) | [钓鱼事件](https://andrej.sh/posts/phishing-through-my-open-source-project)
