# 2026-05-29 RSS 日报

## 今日概览
AI 大模型与代理工具密集更新：Anthropic 正式发布 Claude Opus 4.8，Google 向全美 AI Ultra 订阅者推出自主代理 Gemini Spark，OpenAI 启动生物防御项目 Rosalind Biodefense 并让 Codex 支持 Windows，xAI 开放 grok-build-0.1 编码模型 API。政策方面，加州议会通过游戏保护法案，Robinhood 允许 AI 代理自主交易引发广泛讨论。同时，AI 服务的高昂 Token 成本与职场强制推行 AI 成为社区焦点。

## 重点事件

### 大模型与 AI 产品
- **Claude Opus 4.8 发布**：Anthropic 发布新版本，主要提升判断力、长时间独立工作与代理编码能力，[价格保持不变](https://www.v2ex.com/t/1216353#reply0)。社区反馈其 Token 消耗较快，[少量修复即耗用大量额度](https://www.v2ex.com/t/1216351#reply0)；多起案例显示通过 API 调用时[模型偶尔会自称“Qwen”](https://www.v2ex.com/t/1216588#reply3)，疑为幻觉。
- **Gemini Spark 上线**：谷歌的 24/7 个人 AI 代理已[向美国 Google AI Ultra 订阅者开放](https://x.com/GeminiApp/status/2060405496872579115)，可跨数字生态自主执行任务。同时谷歌[修复了配额消耗过快的问题](https://x.com/GeminiApp/status/2060173327201890662)。
- **OpenAI 多项发布**：启动 [Rosalind Biodefense 项目](https://openai.com/index/strengthening-societal-resilience-with-rosalind-biodefense)，向开发者与政府合作方开放 GPT-Rosalind 用于生物防御与公共卫生；[Codex 计算机使用功能现已支持 Windows](https://x.com/OpenAI/status/2060428604727771421)；发布[第三方前沿 AI 评估指南](https://openai.com/index/trustworthy-third-party-evaluations-foundations)。
- **xAI Grok Build API 公测**：grok-build-0.1 模型[通过 API 公开测试](https://x.com/elonmusk/status/2060451655301132662)，专注智能编码，定价为输入 $1/百万 tokens、输出 $2/百万 tokens；[每日更新日志可在线查看](https://x.ai/build/changelog)。
- **Liquid AI 开源混合专家模型**：[8B-A1B MoE 模型发布](https://www.liquid.ai/blog/lfm2-5-8b-a1b)，基于 38T 数据训练。
- **Kog AI 实时推理**：在标准 GPU 上实现[每请求 3,000 tokens/s 的实时 LLM 推理](https://blog.kog.ai/real-time-llm-inference-on-standard-gpus-3-000-tokens-s-per-request/)。

### 政策与监管
- **加州通过《保护我们的游戏法案》**：加州众议院通过 [Protect Our Games Act](https://www.invenglobal.com/articles/22330/stop-killing-games-movement-gains-momentum-california-assembly-passes-game-protection-bill)，限制游戏发行商在停服后导致玩家无法访问已购游戏的行为。
- **白宫研究拨款新规**：提出规则赋予政治任命官员[对研究拨款的最终批准权](https://www.scientificamerican.com/article/white-house-proposes-new-rules-giving-political-appointees-final-say-on-research-grants/)，引发科研界关注。

### 行业应用与商业动态
- **Robinhood 允许 AI 代理交易股票**：Robinhood 宣布[用户 AI 代理可自主进行股票交易](https://techcrunch.com/2026/05/27/robinhood-now-lets-your-ai-agents-trade-stocks/)，社区激烈讨论其潜在风险。
- **特斯拉 FSD 进入爱沙尼亚**：成为[第三个批准全自动驾驶的欧洲国家](https://x.com/elonmusk/status/2060349039099469896)（继荷兰、立陶宛后）。
- **Shift 公司用免费清洁换训练数据**：初创公司 Shift [为用户免费清洁房屋以收集数据](https://www.theverge.com/ai-artificial-intelligence/939765/ai-training-data-startup-shift-free-cleaning)，用于未来机器人训练。
- **波士顿儿童医院利用 AI 诊断罕见病**：通过 OpenAI 技术[成功诊断超过 40 例罕见病](https://openai.com/index/boston-childrens-hospital)。
- **鸿蒙生态设备超 13 亿**：开源鸿蒙开发者大会公布[生态设备累计出货超 13 亿台，市场份额连续 7 季度领先 iOS](https://www.v2ex.com/t/1216329#reply31)。

### 开发工具与开源
- **iOS 小组件工具 Await**：[用 TypeScript 快速创建可交互小组件](https://sspai.com/post/110106)，无需 Xcode，降低创作门槛。
- **“口袋植物”App 上线**：聚焦[养护日记与植物成长相册](https://sspai.com/post/109936)，源自播客社群反馈。
- **Claude Code Workflow 原理**：[由主模型生成 JS 编排脚本，本地 Runtime 执行](https://www.v2ex.com/t/1216558#reply0)，并非模型间直接并发。
- **开源项目 MobileGym**：纯前端[浏览器安卓仿真系统](https://www.v2ex.com/t/1216344#reply10)，28 个仿真 App，单实例 400 MB 内存，用于 GUI Agent 研究。
- **PAM-OS 开源**：[本地优先 AI 记忆操作系统](https://www.v2ex.com/t/1216374#reply0)，拆离模型记忆层，支持 CLI/API/MCP。
- **Hermes Memory Sidecar v3.0**：[为任意 AI 智能体加装长期记忆](https://www.v2ex.com/t/1216371#reply0)，支持分层召回，已处理超 12,600 会话。
- **GitHub 热门项目**：[anthropics/claude-code](https://github.com/anthropics/claude-code)（终端代理编程）、[DataTalksClub/data-engineering-zoomcamp](https://github.com/DataTalksClub/data-engineering-zoomcamp)（数据工程训练营）、[Crosstalk-Solutions/project-nomad](https://github.com/Crosstalk-Solutions/project-nomad)（离线生存计算机）等。

## 趋势与判断

- **AI Token 成本成为重大瓶颈**：阮一峰周刊援引数据指出，若不限量使用顶级模型，单个程序员每月可[消耗 130 万美元 Token](http://www.ruanyifeng.com/blog/2026/05/weekly-issue-398.html)；Uber 今年前四个月花完全年 34 亿美元 AI 预算；微软因费用超标放弃外部 Claude Code。Google Gemini API 两轮对话即耗费 $4.92，[用户直呼太贵](https://www.v2ex.com/t/1216386#reply1)。
- **AI 代理自主化加速，但信任与风险并存**：从 Gemini Spark 的后台自主执行到 Robinhood 的 AI 交易，代理正在深入关键场景。同时，社区[列出金融行情数据接入的 7 大工程风险](https://www.v2ex.com/t/1216537#reply0)，包括字段漂移、幻觉“编造数字”等，提醒落地需谨慎。
- **职场强制 AI 引发矛盾**：多地出现领导强推 AI、硬性加班学习 AI 的现象，有[研发团队因此被延迟调薪](https://www.v2ex.com/t/1216381#reply14)，员工嘲讽“AI 生成垃圾，员工通宵重做”。同时，开发者[月均消费 1200 元订阅 AI 工具](https://www.v2ex.com/t/1216578#reply12)，产生重度依赖。
- **开源安全问题浮现**：开发者[开源项目被用于网络钓鱼](https://andrej.sh/posts/phishing-through-my-open-source-project)，Turso 因 AI 生成无效漏洞报告泛滥而[终止漏洞赏金计划](http://www.ruanyifeng.com/blog/2026/05/weekly-issue-398.html)。
- **跨端开发框架选择出现分化**：对 24 个已上架 App 的调研显示，[独立开发者中原生（Swift/SwiftUI）使用比例最高](https://www.v2ex.com/t/1216376#reply2)，其次为 Flutter 和 Tauri。
- **模型幻觉与身份混淆**：Claude Opus 4.8 自称 Qwen 的案例，以及 ChatGPT Temporary Chat 的输出逻辑缺陷，表明大模型在可靠性与行为一致性上仍有明显短板。
