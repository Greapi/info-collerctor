# 2026-06-29 RSS 日报

## 今日概览
今日科技领域的关键词是“AI 应用落地”与“产业变局”。Rokid 发布智能眼镜操作系统，试图以 AI 原生交互“删掉 APP”；Cursor 推出 iOS 版，将开发环境延伸至移动端。AI 模型价格战与竞争加剧，DeepSeek V4 宣布峰谷定价，Grok 4.5 内部测试进展曝光。商业层面，存储芯片三巨头在美被诉、Rocket Lab 收购铱星等事件，反映出科技产业的结构性调整正在加速。

## 重点事件
- **Rokid 发布 YodaOS，定义智能眼镜新范式**：Rokid 在杭州将 YodaOS 定义为“全球首款智能眼镜 AIOS”，核心交互是用户直接向 AI 提需求、系统调用服务呈现结果，宣称要“删掉 APP”。其 AIUI 框架可自动生成富内容（如 K 线图），同时发布了乐奇 AI 助手 2.0，并成为首个同时接入微信扫一扫和支付宝两大支付体系的 AI 眼镜。生态方面，智能体商店已上架超 440 个智能体，开发者社区注册人数超 3.3 万。硬件上发布了新一代 Rokid AR（分体式设计，Micro-OLED + 混合波导，FOV 58°）及 Q4 将发售的 AI 头盔。创始人判断有效用户突破 200 万台后，C 端开发者才能真正涌入。[来源](https://sspai.com/post/111701)

- **Cursor 发布 iOS 版本**：Cursor for iOS 允许用户通过启动常驻云代理从任何地方构建项目，或远程控制电脑上运行的代理。Composer 2.5 在应用内提供 75% 折扣，持续到 7 月 5 日。[来源](https://x.com/elonmusk/status/2071650759330906253)

- **DeepSeek V4 即将实行峰谷定价**：`deepseek-v4-pro` 模型百万 tokens 输出价格在高峰时段（北京时间 9:00-12:00， 14:00-18:00）从 6 元涨至 12 元，`flash` 版本从 2 元涨至 4 元。[来源](https://www.v2ex.com/t/1223737#reply0)

- **Grok 4.5 内部测试进展曝光**：基于 1.5T 参数的 V9 基础模型训练，加入 Cursor 数据做补充训练，已在 SpaceX 与特斯拉内部测试。初步评估性能接近甚至可能超越 Opus。[来源](https://x.com/elonmusk/status/2071184354756477041)

- **存储芯片三巨头在美被诉价格操纵**：三星、SK 海力士、美光在美国被起诉，涉嫌操纵存储芯片价格。[来源](https://en.sedaily.com/international/2026/06/29/samsung-sk-hynix-micron-sued-in-us-over-memory-price-fixing) 与此形成对比的是，美光最新 FQ3-26 财报毛利率达 84.9%，刷新商业史纪录。[来源](https://www.v2ex.com/t/1223766#reply0)

- **Rocket Lab 宣布收购铱星通信公司**：该收购旨在创建一家完全垂直整合的航天公司。[来源](https://investors.rocketlabcorp.com/news-releases/news-release-details/rocket-lab-acquire-iridium-historic-deal-creating-fully)

- **美国最高法院裁定地理围栏搜查令需符合宪法保护**：该裁决增加了执法部门使用此类技术的法律限制。[来源](https://www.theguardian.com/us-news/2026/jun/29/supreme-court-geofence-warrants-case-decision)

- **HP Inc. 与 OpenAI 建立前沿战略伙伴关系**：HP 将 AI 部署于客户体验、软件开发及公司运营。[来源](https://openai.com/index/hp-frontier-partnership)

- **谷歌限制 Meta 使用其 Gemini AI 模型**：CNBC 报道谷歌限制 Meta 使用其 Gemini AI 模型。[来源](https://www.cnbc.com/2026/06/28/google-limits-metas-use-of-its-gemini-ai-models-ft-reports.html)

- **特斯拉向 AI3 硬件推送 FSD V14 Lite**：该版本将 HW4 V14 的智能“蒸馏”到 AI3 硬件上，更新包括提升主动响应能力、改善舒适性、引入泊车与倒车功能以及新增到达选项和速度模式等。[来源](https://x.com/elonmusk/status/2071606213456859284)

## 趋势与判断
- **AI 原生硬件的新入口之争**：Rokid 提出的“删掉 APP”理念与 Cursor 的移动端延伸，都试图在传统智能手机之外建立 AI 原生交互入口。智能眼镜作为载体若能突破 200 万台用户量，或将催生独立于手机的应用生态。[来源](https://sspai.com/post/111701)

- **大模型 API 价格内卷加剧，峰谷定价出现**：继 DeepSeek V4 的峰谷调价之后，市场上各类 API 聚合站、中转站大量涌现，折扣低至官方费率的 0.3 倍甚至 0.06 倍。这一方面降低了开发者成本，另一方面也指向了模型同质化竞争加剧、各家试图通过价格手段锁定用户的激烈态势。[来源](https://www.v2ex.com/t/1223737#reply0) [来源](https://www.v2ex.com/t/1223480#reply25)

- **AI 与开发工具深度耦合，职业范式加速转变**：AI 工程概念从 Prompt Engineering 到 Context Engineering、Harness Engineering，再到 Loop Engineering 的演进脉络引发开发者热议。用户开始实践“用 Claude 调度 Codex 和 Agony”等复杂工作流，“Agent 看板”等监控工具也随之出现。但同时，Claude Code 的严重幻觉和 Codex/Claude 巨大的 token 消耗统计口径差异，也揭示出产品稳定性仍需提升。[来源](https://www.v2ex.com/t/1223448#reply17) [来源](https://www.v2ex.com/t/1223775#reply3)

- **全球科技产业的结构性整合**：Rocket Lab 收购铱星、存储芯片三巨头被诉，以及 Apollo 报告指出“Mag 7”（七大科技巨头）股票开始表现不佳，这些事件共同指向一个趋势——科技产业正从野蛮生长走向垂直整合与存量竞争，格局变动期可能来临。[来源](https://en.sedaily.com/international/2026/06/29/samsung-sk-hynix-micron-sued-in-us-over-memory-price-fixing) [来源](https://investors.rocketlabcorp.com/news-releases/news-release-details/rocket-lab-acquire-iridium-historic-deal-creating-fully) [来源](https://www.apollo.com/content/dam/apolloaem/pdf/daily-spark/2026/jun/28/062826-Mag7.pdf)
