# 2026-07-14 RSS 日报

## 今日概览
今日 AI 领域围绕 Grok 4.5 展开密集讨论，该模型在多项基准测试中登顶，并因速度与性价比获得社区好评。同时，Grok Build 的代码上传隐私争议仍在发酵。OpenAI 方面，Sam Altman 宣布 Agent 产品用量周增长 2.5 倍，但用户社区反馈 GPT-5.6 模型出现速度变慢、高峰卡顿等问题。行业层面，Anthropic 宣布教育项目与加拿大学术投资，Oracle 信用评级被下调至接近垃圾级。

## 重点事件

### Grok 4.5 登顶多项基准测试
Elon Musk 转发了 Grok 4.5 在“长程终端基准测试（Long-Horizon Terminal-Bench）”中排名第一的成绩。该基准论文指出，此前测试的 15 个模型平均仅完成约 2 项任务，最佳者仅完成 7 项。([来源](https://x.com/elonmusk/status/2076909201524203973))
同时，Grok 4.5 在 SWE-Atlas-QnA 基准测试中得分最高，超越了 Claude Fable 5 与 GPT-5.6 Sol。([来源](https://x.com/elonmusk/status/2076538528041677013))

### Grok Build 数据隐私争议
Grok Build 被发现在运行过程中将用户代码打包上传至谷歌云存储，经社交媒体发酵后，SpaceXAI 官方回应称通过 `/privacy` 命令可更改设置并删除先前同步数据。([来源](https://www.v2ex.com/t/1227071#reply0)) Elon Musk 此前表示，所有此前上传至 SpaceXAI 的用户数据将被完全删除。([来源](https://x.com/elonmusk/status/2076739687658496209))

### OpenAI Agent 产品用量爆发，但 GPT-5.6 遭遇性能争议
OpenAI CEO Sam Altman 宣布，其智能体产品 Codex 和 ChatGPT Work 的用量在过去一周增长了 **2.5 倍**，同时 GPT-5.6 Sol 价格仅为 Fable 的一半，Token 效率约为 Fable 的两倍。([来源](https://x.com/sama/status/2077033807736459713)、[来源](https://x.com/sama/status/2077036999303999910))
但用户社区普遍反映 GPT-5.6 速度变慢，高峰时段卡顿严重，有用户称 Sol Ultra 模型一轮对话 25 分钟即消耗 Plus 周限额的 20%。([来源](https://www.v2ex.com/t/1227026#reply1))

### OpenAI 调整产品线
ChatGPT 的 Agent Mode 入口被移除，Codex 被调整为 ChatGPT，原 ChatGPT 更名为 ChatGPT Classic。([来源](https://www.v2ex.com/t/1227332#reply0)、[来源](https://www.v2ex.com/t/1227276#reply2))

### Anthropic 推出 Claude 教育服务与学术投资
Anthropic 宣布推出“Claude for Teachers”项目，为美国认证的 K-12 教育工作者免费提供高级 Claude 功能。([来源](https://x.com/AnthropicAI/status/2077047619260707263)) 同日，该公司承诺投入 1000 万加元与加拿大 AI 机构合作资助新研究。([来源](https://x.com/AnthropicAI/status/2077026346375540870))

### Oracle 信用评级下调
标准普尔将 Oracle 信用评级下调至 **BBB**，仅比垃圾级高一个等级。([来源](https://www.heise.de/en/news/S-P-downgrades-Oracle-to-BBB-only-one-notch-above-junk-level-11363472.html))

### SpaceX 与星链商业进展
SpaceX 利用猎鹰 9 号从加州发射 27 颗星链卫星。([来源](https://x.com/elonmusk/status/2077043064099483926)) 星链宣布将为边疆航空提供机上高速连接服务，预计 2027 年启用。([来源](https://x.com/elonmusk/status/2077042568907391273))

### 欧洲政治风向
马斯克转发的帖文指出欧洲右翼浪潮兴起，法国勒庞、奥地利自由党、德国 AfD 及挪威 FrP 均在民调中大幅领先。([来源](https://x.com/elonmusk/status/2077042177520111682))

## 趋势与判断

### “性价比模型”时代加速到来
Grok 4.5 被社区评价为比 Fable 5 快且便宜 90%，且成功完成了 Fable 5 无法处理的任务（45 万个文件索引搜索）。([来源](https://x.com/elonmusk/status/2077042663946113337)) GPT-5.6 Sol 同任务总价仅为 Fable 四分之一。([来源](https://x.com/sama/status/2077036999303999910)) 同时，AI 模型代际更替极快——18 个月前被认为革命性的 DeepSeek V3 如今已基本不可用。([来源](https://x.com/elonmusk/status/2076907660784353785))

### AI 隐私与数据安全风险成为行业焦点
Grok Build 代码上传事件暴露出 AI 开发工具在数据保留策略上的模糊地带。([来源](https://www.v2ex.com/t/1227071#reply0)) 企业微信文档也被曝存在重大安全漏洞。([来源](https://www.v2ex.com/t/1227287#reply1)) 行业对“零数据保留”政策的信任正在经受考验。

### 智能体交互范式转向“Heredoc 式执行”
有研究指出，相比 REPL 式交互，Heredoc 式执行更适合 Agent。通过其 ego-browser 项目实验，可将平均执行轮数、工具调用次数和成本降低约 40%，Heredoc 成功率达 77.5%，Token 消耗减少 29.8%。([来源](https://www.v2ex.com/t/1227024#reply0))

### 加密货币合规化信号
Coinbase 已支持中国大陆身份注册，可使用 86 手机号和护照或身份证完成 KYC 认证，被视为向 Kraken 看齐的合规化动向。([来源](https://www.v2ex.com/t/1227325#reply4))
